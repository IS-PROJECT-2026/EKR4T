# Hi there, I'm Emmanuel Keter 👋
I’m a passionate software developer and problem-solver focused on building intelligent, data-driven systems that create real-world impact. I enjoy working at the intersection of AI, fintech, and web technologies, especially developing solutions that empower communities like smallholder farmers. My goal is to design scalable, user-centric applications that bridge the gap between technology and practical everyday challenges.


🔭 **Current Focus:** Building FarmScoreAI (a USSD-based credit scoring system) and FarmIQ (an AI-powered platform with personalized insights), while also developing a RAG chatbot (FarmGrow) using OCR and LLMs.


🌱 **Learning & Exploration:** Diving deeper into AI/ML integration, LLM optimization (Llama 3), Supabase backends, Angular PWAs, and advanced prompt engineering for smarter, context-aware applications.


⚡ **Core Interests:** Artificial Intelligence, Fintech, Agritech, Web Development, Data Systems, and building scalable cloud-powered solutions.
